🧪 Test Rezervačního Tlačítka

📅 Kalendář s Rezervačním Tlačítkem

Tento test ověřuje funkčnost rezervačního tlačítka, které se zobrazí po výběru termínu.

💡 Jak testovat

1. Vyberte datum příjezdu kliknutím na dostupný den

2. Vyberte datum odjezdu kliknutím na další dostupný den

3. Pod kalendářem se zobrazí rezervační tlačítko s cenou

4. Klikněte na rezervační tlačítko - mělo by otevřít novou záložku s URL obsahující parametry

🔗 Očekávané URL parametry

Rezervační tlačítko by mělo generovat URL s následujícími parametry:

https://booking.example.com?checkin=YYYY-MM-DD&checkout=YYYY-MM-DD&guests=2¤cy=CZK

Parametry URL:

  • checkin - Datum příjezdu ve formátu YYYY-MM-DD
  • checkout - Datum odjezdu ve formátu YYYY-MM-DD
  • guests - Počet osob (z konfigurace)
  • currency - Měna (z konfigurace)
  • unit - ID jednotky (pokud je specifikováno)

✅ Testovací scénáře

Doporučené testy:

  1. Základní funkce - Vyberte jakýkoliv dostupný termín a klikněte na rezervační tlačítko
  2. Různé délky pobytu - Otestujte pobyty různých délek (1 noc, 3 noci, týden)
  3. Přes hranici měsíce - Vyberte termín přesahující do dalšího měsíce
  4. Různé měny - Změňte měnu v demo ovládacích prvcích a ověřte, že se správně předává
  5. Různý počet osob - Změňte počet osob a ověřte parametr guests
  6. Lokalizace - Přepněte mezi češtinou a angličtinou a ověřte text tlačítka

🐛 Známé problémy

Co kontrolovat:

  • Rezervační tlačítko se zobrazuje pouze po výběru obou dat
  • URL se generuje správně s aktuálními daty
  • Tlačítko otevírá novou záložku (target="_blank")
  • Text tlačítka obsahuje správnou cenu a měnu
  • Styly tlačítka jsou správně aplikovány
